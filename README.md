# Mobile App Project

## Mô tả

Dự án ứng dụng di động.

## Yêu cầu

- Node.js >= 18
- npm hoặc yarn

## Cài đặt

```bash
# Clone repo
git clone <url-repo-của-bạn>

# Di chuyển vào thư mục dự án
cd mobile-app-project

# Cài đặt dependencies
npm install
```

## Chạy ứng dụng

```bash
npm start
```

## Cấu trúc thư mục

```
mobile-app-project/
├── src/
│   ├── assets/          # Hình ảnh, fonts, icons
│   ├── components/      # Các component dùng chung
│   ├── screens/         # Các màn hình chính
│   ├── navigation/      # Cấu hình điều hướng
│   ├── utils/           # Hàm tiện ích
│   └── styles/          # Style dùng chung
├── .gitignore
├── README.md
└── package.json
```
